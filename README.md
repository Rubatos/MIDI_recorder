# Simple MIDI Recorder
 :notes: MIDI recorder (support program for the [CodeKlavier](https://codeklavier.space) research phase)

## instructions

- make sure you have python3 installed
- pip3 install rtmidi
- pip3 install mido

###### then simply run in terminal:

`python3 recorder.py`
